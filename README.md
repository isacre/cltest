## CodeLeap Test

This is a test project for CodeLeap.

## Hosted on Netlify

[https://isaac-codeleap.netlify.app/](https://isaac-codeleap.netlify.app/)

## Running Locally 

1. Clone the repository

2. Install dependencies

```bash
npm install
```
3. Create a .env file based on the .env.example file

4. Run the development server
```bash
npm run dev
```

5. Run the tests
```bash
npm run test
```
